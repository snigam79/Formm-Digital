01 — Visual System
The new design language
Warm-dark, precision-data, sports-science credibility. Inspired by elite performance software, not generic wellness apps.
FOUNDATION
Current look
Pure cold black (#0A0A0A) — no depth
Generic Inter/system font — no brand
Flat equal-weight cards everywhere
Cyan-blue (#00C4A0) — too generic
No gradient, no texture, no atmosphere
All borders same weight (1px)
Colour used decoratively, not semantically
New look
Warm dark (#08080C) — subtle warmth
Clash Display + Cabinet Grotesk — distinctive
Layered card hierarchy (bg2/bg3/bg4)
Electric teal (#00E0B5) — sharper, ownable
Radial glow accents on hero cards
Variable border opacity (ln1/ln2/ln3)
Colour strictly semantic (green=safe, amber=caution)
💡
Design philosophy: The visual language says "elite sports science tool" not "wellness app." Inspired by Catapult, Whoop, and Bloomberg Terminal — precision data presented with clarity and weight. Every visual decision earns trust before the user reads a word.

Yes, what's the fastest way to improve depth?
Almost always an ankle mobility issue, not strength. Try this before squatting:

1. 10 ankle circles each direction, both sides
2. Wall ankle stretch — toe 5cm from wall, knee to wall without heel lifting, 3×10 per side
3. Goblet squat with 5-second pause at the bottom

Do this for 2 weeks and you'll likely add 3–5cm of depth.
